Pie Chart Representation¶

fatalities_per_operator.plot(kind='pie', autopct='%1.1f%%', figsize=(12,6)): Creates a pie chart of the 'fatalities_per_operator' data, with each operator's percentage of fatalities represented as a slice of the pie. The 'kind' parameter is set to 'pie' to create a pie chart. The 'autopct' parameter specifies the format of the percentage values displayed on the chart. The 'figsize' parameter sets the size of the chart.

In [ ]:
# Import libraries
import pandas as pd
import matplotlib.pyplot as plt

# Mount Google Drive to access the dataset
from google.colab import drive
drive.mount('/content/drive')

# Load data
data = pd.read_csv('/content/Airplane_Crashes_and_Fatalities_Since_1908.csv')

# Create a pie chart of the number of fatalities per operator
fatalities_per_operator = data.groupby('Operator').sum()['Fatalities'].nlargest(10)
fatalities_per_operator.plot(kind='pie', autopct='%1.1f%%', figsize=(12,6))
plt.title('Percentage of Fatalities per Top 10 Operators')
plt.ylabel('')
plt.show()

Scatter Plot Representation¶

This code uses the Python libraries Pandas and Matplotlib to create a scatter plot of the number of fatalities versus the number of people aboard for airplane crashes recorded in a dataset.

  • First, Pandas is used to load the dataset Airplane_Crashes_and_Fatalities_Since_1908.csv into a DataFrame named data.

  • Next, a scatter plot is created using the plot() method on the data DataFrame. The kind parameter is set to 'scatter' to specify that a scatter plot should be created. The x parameter is set to 'Aboard', which is the column in the DataFrame containing the number of people aboard the airplane. The y parameter is set to 'Fatalities', which is the column in the DataFrame containing the number of fatalities resulting from the airplane crash. The figsize parameter is set to (12,6) to specify the size of the plot.

In [ ]:
# Import libraries
import pandas as pd
import matplotlib.pyplot as plt

# Load data
data = pd.read_csv('/content/Airplane_Crashes_and_Fatalities_Since_1908.csv')
# Create a scatter plot of the number of fatalities vs. the number of aboard
fatalities_vs_aboard = data.plot(kind='scatter', x='Aboard', y='Fatalities', figsize=(12,6))
plt.title('Number of Fatalities vs. Number of Aboard')
plt.xlabel('Number of Aboard')
plt.ylabel('Number of Fatalities')
plt.show()

Line Chart Representation¶

  • The data is loaded from the 'Airplane_Crashes_and_Fatalities_Since_1908.csv' file using the Pandas library.
  • The data is grouped by 'Time' (year of crash) using the groupby() method and the size() method is used to count the number of crashes per year.
  • The resulting data is plotted as a line chart using the plot() method with the 'line' kind parameter.
  • The chart has a title 'Number of Airplane Crashes per Year', x-label 'Year', and y-label 'Number of Crashes'.
In [ ]:
# Import libraries
import pandas as pd
import matplotlib.pyplot as plt

# Load data
data = pd.read_csv('/content/Airplane_Crashes_and_Fatalities_Since_1908.csv')

# Create a line chart of the number of crashes per year
crashes_per_year = data.groupby('Time').size()
crashes_per_year.plot(kind='line', figsize=(12,6))
plt.title('Number of Airplane Crashes per Year')
plt.xlabel('Year')
plt.ylabel('Number of Crashes')
plt.show()

Examples
--------

    The simplest way to use nbconvert is

            > jupyter nbconvert mynotebook.ipynb

            which will convert mynotebook.ipynb to the default format (probably HTML).

            You can specify the export format with `--to`.
            Options include ['asciidoc', 'custom', 'html', 'latex', 'markdown', 'notebook', 'pdf', 'python', 'rst', 'script', 'slides'].

            > jupyter nbconvert --to latex mynotebook.ipynb

            Both HTML and LaTeX support multiple output templates. LaTeX includes
            'base', 'article' and 'report'.  HTML includes 'basic' and 'full'. You
            can specify the flavor of the format used.

            > jupyter nbconvert --to html --template basic mynotebook.ipynb

            You can also pipe the output to stdout, rather than a file

            > jupyter nbconvert mynotebook.ipynb --stdout

            PDF is generated via latex

            > jupyter nbconvert mynotebook.ipynb --to pdf

            You can get (and serve) a Reveal.js-powered slideshow

            > jupyter nbconvert myslides.ipynb --to slides --post serve

            Multiple notebooks can be given at the command line in a couple of 
            different ways:

            > jupyter nbconvert notebook*.ipynb
            > jupyter nbconvert notebook1.ipynb notebook2.ipynb

            or you can specify the notebooks list in a config file, containing::

                c.NbConvertApp.notebooks = ["my_notebook.ipynb"]

            > jupyter nbconvert --config mycfg.py

To see all available configurables, use `--help-all`.
